HTML을 공부하면서 **메타 태그(meta tag)**에 대해 다시 한번 정리해 보았다.
메타 태그는 쉽게 말해 HTML 문서의 **메타데이터(문서에 대한 정보)**를 담는 태그다.

보통 <head> 안에 작성되며 직접 화면에 표시되지는 않는다. 하지만 브라우저, 검색엔진, SNS가 페이지를 이해하는 데 중요한 역할을 한다.

대표적으로 자주 쓰이는 메타 태그는 다음과 같다.

- **charset** → 문서의 문자 인코딩 방식을 지정한다. 보통 UTF-8을 사용한다.

- **viewport** → 반응형 웹을 만들 때, 모바일 화면 크기에 맞게 조정하는 핵심 태그다.

- **description** → 검색엔진 결과에 노출되는 페이지 설명을 제공한다.

- **Open Graph 태그(og:title, og:image 등)** → SNS에서 페이지를 공유할 때 보이는 제목, 이미지, 설명을 제어한다.

정리하자면, 메타 태그는 페이지가 어떻게 해석되고, 검색되고, 공유될지를 결정하는 중요한 정보 제공자다. 앞으로 프로젝트를 할 때 <head> 영역을 그냥 지나치지 않고, 꼭 필요한 메타 태그들을 꼼꼼히 챙겨 넣어야겠다고 생각했다.-
